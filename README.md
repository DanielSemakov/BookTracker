The BookTracker app allows users to keep track of the books they have read. From the home page, the user can either register with a new account or login with an existing account.
From there, they can select buttons which allow them to modify their book collection by adding a book, viewing all books, updating a book's info, or deleting a book. When 
adding a book to the collection, the user should specify the book's ISBN, name, author, genre, publisher and rating. When updating a book's info, the user should identify
the book by its ISBN. Once the book is identified, all the fields except for the ISBN may be edited. Every time a change is made to the book collection, the user must select the 
"View All Books" button in order to view the updated collection. A user may also select the Logout button to return to the home page. When a user logs out, all their book data is 
saved to their account so long as the program continues running. When the program terminates, all data is cleared.
